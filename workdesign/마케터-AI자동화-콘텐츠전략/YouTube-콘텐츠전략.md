## 2. 영상 구조 패턴

1. **첫 5초** — 고통점 직접 제시 (이탈 여부 여기서 결정)
2. **영상 길이** — 6시간 풀코스(875K뷰)도 터짐. 길이보다 밀도가 중요
3. **CTA** — 영상 안에서 최소화, 설명란에만 링크 (Nick Saraev 월 $300K+ 구조)
4. **무료 템플릿 제공** — 즉각적 가치 → 구독/클릭 유도

## 3. 퍼널 구조 (단순할수록 이김)

```
YouTube 영상
    ↓
설명란 링크 하나 (웨이팅리스트 or 커뮤니티)
    ↓
얼리엑세스 유료 전환
```

- 복잡한 퍼널 없음, 영상 내 CTA 최소화

## 6. 수요 검증 → 런칭 퍼널 (SetterAI 패턴)

```
데모 영상 제작 (제품 없어도 됨)
    ↓
YouTube / 인스타 / X 게시 + CTA: "관심 있으면 댓글/DM"
    ↓
반응 확인 → 있으면 SaaS 개발 시작
    ↓
얼리엑세스 유료 (무료 플랜 없이)
    ↓
고객 피드백 → 기능 추가 반복
```
