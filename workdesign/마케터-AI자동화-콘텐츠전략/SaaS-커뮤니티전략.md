## 핵심 구분

| | 강의 커뮤니티 (Skool 모델) | SaaS 커뮤니티 |
|--|--|--|
| 수익 모델 | 커뮤니티 멤버십이 곧 수익 | 제품 구독이 수익, 커뮤니티는 도구 |
| 역할 | 콘텐츠 판매 채널 | 제품 확산 + 리텐션 엔진 |
| 예시 | Nick Saraev $184/월, Liam Ottley | n8n, Figma, Notion, Zapier |

---

## 실제 SaaS 커뮤니티 퍼널

```
YouTube 데모 영상
    ↓
무료 체험 (제품 직접 써봄)
    ↓
Discord / Discourse 커뮤니티 진입
  → 질문하고, 템플릿 받고, 다른 유저 사례 보고
  → 제품 의존도 자연스럽게 상승
    ↓
무료 플랜 한계 체감 (실행 횟수, 기능 제한)
    ↓
유료 플랜 결제
```

### n8n (워크플로우 자동화)
- 플랫폼: 자체 Discourse 포럼 (community.n8n.io)
- 커뮤니티 멤버: 30,000+명
- 전체 활성 사용자: 230,000+명
- ARR: $40M (2025년)
- **핵심:** 커뮤니티 멤버가 노드 5,834개 직접 제작 → 제품 기능이 공짜로 확장
- 출처: [n8n 2024 Annual Review](https://blog.n8n.io/2024-in-review/)

### Figma (디자인 SaaS)
- 커뮤니티 플러그인: 1,000개+
- MAU: 1,300만+명
- 시장점유율: 7%(2017) → 90%(2023)
- **핵심:** 4년간 무료만 운영, 커뮤니티가 유료화 이전 제품 확산 담당
- 출처: [First Round Capital 케이스 스터디](https://review.firstround.com/the-5-phases-of-figmas-community-led-growth-from-stealth-to-enterprise/)

- 사용자: 1억명+
- 공유 템플릿: 400만+개
- **핵심:** 유저가 템플릿 제작/공유 → 커뮤니티가 마케팅 비용 대체
- 출처: [Bettermode - Notion 커뮤니티 성장](https://bettermode.com/blog/notion-community-led-growth)

### Webflow (노코드 웹빌더)
- 포럼 멤버: 75,000명+
- 매출 YoY: +66% (2024년)

## 초기 SaaS 커뮤니티 실행 순서

커뮤니티를 먼저 만드는 게 아니라, **제품이 커뮤니티를 만들게 하는 것**이 핵심.

```
① 데모 영상 → 얼리어답터 DM/댓글 수집
② 얼리어답터 소수 (10~30명) Discord 채널 하나
   → 피드백 받아 기능 추가
   → 첫 유료 결제자로 전환
③ 제품 쓸수록 공유하고 싶어지는 구조 설계
   (공유 링크에 브랜딩, 템플릿 공유 등)
④ 자연스럽게 커뮤니티 형성
```
