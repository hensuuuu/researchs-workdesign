DAY 1

사진 → 엑셀 변환

VSCode + Claude로 이미지 속 텍스트를 엑셀 파일로 만들기

진행률 0 / 6 완료
1
작업 폴더 만들기
  • 1바탕화면 빈 곳에서 오른쪽 클릭
  • 2"새로 만들기" → "폴더" 클릭
  • 3폴더 이름에 ai-작업 입력 → Enter
바탕화면에 ai-작업 폴더가 생겼으면 성공
2
사진 파일 폴더에 넣기
  • 1정리할 사진 파일 찾기 (명함, 영수증, 문서 등)
  • 2사진 파일 오른쪽 클릭 → "복사"
  • 3ai-작업 폴더 더블클릭으로 열기
  • 4폴더 안 빈 곳 오른쪽 클릭 → "붙여넣기"
ai-작업 폴더 안에 사진 파일이 보이면 성공
3
VSCode에서 폴더 열기
  • 1VSCode 실행
  • 2상단 메뉴 File 클릭
  • 3Open Folder 클릭
  • 4ai-작업 폴더 선택 → "폴더 선택" 버튼 클릭
VSCode 왼쪽 사이드바에 사진 파일 이름이 보이면 성공
4
Claude에 사진 첨부하기
  • 1VSCode 왼쪽 사이드바에서 Claude 아이콘 클릭
  • 2Claude 채팅창이 열리면 성공
  • 3채팅창 하단 클립 아이콘(📎) 클릭 → 사진 파일 선택

📌 클립 아이콘 없으면 — 사진 파일을 채팅창으로 드래그해서 끌어다 놓기

채팅창에 사진 미리보기가 뜨면 성공
5
Claude에게 요청하기

사진이 첨부된 상태에서 아래 문장을 복사해서 붙여넣기

이 사진에서 텍스트를 전부 읽어줘. 읽은 내용을 표 형식으로 정리해서 result.xlsx 파일로 저장해줘. 파일은 지금 열려있는 폴더(ai-작업)에 저장해줘.

→ Enter 또는 전송 버튼 클릭

6
결과 확인
  • 1VSCode 왼쪽 사이드바에 result.xlsx 파일이 생겼는지 확인
  • 2result.xlsx 더블클릭 → 엑셀로 열림
  • 3내용이 들어있으면 성공 🎉
엑셀 파일이 열리고 내용이 있으면 Day 1 완료
