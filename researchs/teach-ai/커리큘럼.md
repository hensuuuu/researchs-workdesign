# 신입사원 AI 실무 커리큘럼

> 대상: 비개발자 신입, 이해력 낮음, 반복 필수
> 환경: 윈도우 + VSCode + Claude Code
> 목표: AI로 크롤링→엑셀, OCR→파일정리 실전 완성

---

## 설계 원칙

1. **클릭 하나하나 명시** — "어디를 누른다"까지 적는다
2. **막히면 대안 제시** — API 안되면 DOM, DOM 안되면 수동
3. **10번 반복 구조** — 한 파트 완전히 익힐 때까지 넘어가지 않는다
4. **실패도 커리큘럼** — 보안 서버 팅기는 경험, 에러 보는 경험 자체가 학습

---

## 전체 구조

```
Phase 0 — 환경 세팅 (1~2일)
Phase 1 — 터미널 + 폴더 개념 (2~3일)
Phase 2 — Claude Code 기본 (2~3일)
Phase 3 — 실전 A: 크롤링 → 엑셀 (1~2주)
Phase 4 — 실전 B: OCR → 파일정리 (1주)
Phase 5 — GitHub + Vercel (나중에)
```

## Phase 0 — 환경 세팅

### 0-1. VSCode 설치
1. 구글에 "VSCode 다운로드" 검색
2. 파란 버튼 "Download for Windows" 클릭
3. 다운로드된 파일 더블클릭 → 설치 (Next 계속 누르면 됨)
4. 설치 완료 후 VSCode 실행

### 0-2. Claude Code 설치
1. VSCode 왼쪽 사이드바에서 네모 4개 아이콘 클릭 (Extensions)
2. 검색창에 "Claude" 입력
3. "Claude" 항목 클릭 → Install 버튼 클릭
4. 설치 완료 후 왼쪽에 Claude 아이콘 생기는지 확인

### 0-3. Node.js 설치 (터미널 명령어 쓰려면 필요)
1. 구글에 "Node.js 다운로드" 검색
2. "LTS" 버전 다운로드 클릭
3. 설치 파일 더블클릭 → Next 계속 → Install
4. 설치 확인: VSCode 터미널 열고 `node -v` 입력 → 버전 숫자 나오면 성공

### 0-4. Python 설치 (크롤링, OCR에 필요)
1. 구글에 "Python 다운로드" 검색
2. 노란 버튼 "Download Python 3.x.x" 클릭
3. **중요: 설치 화면 맨 아래 "Add Python to PATH" 체크박스 반드시 체크**
4. Install Now 클릭
5. 확인: VSCode 터미널에 `python --version` 입력 → 버전 나오면 성공

## Phase 1 — 터미널 + 폴더 개념

### 왜 배우는가
Claude Code는 터미널에서 돌아간다.
터미널을 모르면 Claude Code를 못 쓴다.

### 1-1. 터미널 여는 법
**VSCode에서 터미널 열기:**
1. VSCode 상단 메뉴에서 "Terminal" 클릭
2. "New Terminal" 클릭
3. 화면 하단에 검은 창 뜨면 성공

> 이 검은 창이 터미널이다. 여기에 명령어를 입력한다.

### 1-2. 폴더 트리 개념
컴퓨터 안의 폴더 구조는 나무처럼 생겼다.
```
C:/
  Users/
    내이름/
      Documents/
        프로젝트/
          파일.py
          파일.xlsx
```
- 폴더 안에 폴더가 있고, 폴더 안에 파일이 있다
- 터미널은 항상 "지금 어느 폴더에 있는지"가 중요하다

### 1-3. 필수 명령어 5개

**지금 어디 있는지 확인:**
```
pwd
```
→ 현재 폴더 경로가 나온다

**폴더 안에 뭐가 있는지 보기:**
```
ls
```
→ 파일/폴더 목록이 나온다

**폴더 이동:**
```
cd 폴더이름
```
예시: `cd Documents` → Documents 폴더로 이동

**한 단계 위로 이동:**
```
cd ..
```

**새 폴더 만들기:**
```
mkdir 폴더이름
```
예시: `mkdir 내프로젝트` → "내프로젝트" 폴더 생성

### 1-4. 실습 (10회 반복)

아래를 터미널에서 직접 해본다:

```
1. 터미널 열기
2. pwd 입력 → 현재 위치 확인
3. cd Documents 입력 → Documents로 이동
4. pwd 입력 → 위치 바뀐 것 확인
5. mkdir ai-실습 입력 → 폴더 생성
6. ls 입력 → ai-실습 폴더 보이는지 확인
7. cd ai-실습 입력 → 폴더 안으로 이동
8. pwd 입력 → 현재 위치 확인
9. cd .. 입력 → 한 단계 위로
10. pwd 입력 → 원래 위치로 돌아왔는지 확인
```

**10회 반복. 막히면 처음부터.**

### 1-5. VSCode에서 폴더 열기
1. VSCode 상단 메뉴 "File" 클릭
2. "Open Folder" 클릭
3. 아까 만든 "ai-실습" 폴더 선택
4. 왼쪽 사이드바에 폴더 내용이 보이면 성공

## Phase 2 — Claude Code 기본

### 2-1. Claude Code란
- 터미널에서 Claude AI를 쓰는 도구
- 파일을 직접 읽고, 코드를 짜고, 실행까지 한다
- 웹 Claude와 다르다 — 내 컴퓨터 파일에 직접 접근한다

### 2-2. Claude Code 시작하기
1. VSCode 터미널 열기
2. 프로젝트 폴더로 이동: `cd Documents/ai-실습`
3. `claude` 입력 → Enter
4. Claude Code 실행되면 성공

### 2-3. 첫 대화
Claude Code가 켜진 상태에서:
```
"안녕하세요, 지금 이 폴더에 있는 파일 목록을 알려주세요"
```
입력 → Claude가 폴더를 직접 보고 답한다

### 2-4. 파일 만들어보기
Claude Code에게:
```
"test.txt 파일을 만들고 안에 '첫 번째 파일입니다' 라고 써주세요"
```
→ 왼쪽 VSCode 사이드바에 test.txt가 생기면 성공

### 2-5. 파일 내용 보기
VSCode 왼쪽 사이드바에서:
1. test.txt 파일 클릭
2. 오른쪽 화면에 내용이 보이면 성공

### 2-6. 실습 (반복)
Claude Code에게 아래를 순서대로 시켜본다:
```
1. "폴더 안에 파일이 몇 개 있어요?"
2. "hello.py 파일 만들어주세요"
3. "hello.py 안에 print('안녕하세요') 코드 넣어주세요"
4. "hello.py 실행해주세요"
5. "터미널에 '안녕하세요'가 출력되면 성공이에요"
```

## Phase 3 — 실전 A: 크롤링 → 엑셀

### 3-0. 크롤링이란
웹사이트에서 데이터를 자동으로 가져오는 것.
예: 네이버 쇼핑 가격 목록을 자동으로 엑셀에 정리

### 3-1. 방법 3가지 (순서대로 시도)

```
방법 1: API 방식 (가장 쉽고 안정적)
  → 사이트가 데이터를 API로 제공하면 이걸 쓴다
  → 막히면 방법 2로

방법 2: DOM 방식 (HTML 직접 파싱)
  → 사이트 HTML에서 원하는 부분 추출
  → 보안 막히거나 팅기면 방법 3으로

방법 3: 수동 복붙
  → 자동화 안되면 직접 복사해서 Claude한테 정리시킨다
```

### 3-2. 실습: 네이버 뉴스 제목 가져오기

**Claude Code에게:**
```
"네이버 뉴스 메인 페이지에서 뉴스 제목 10개를 가져와서
news.xlsx 파일에 정리해주세요.
먼저 API 방식으로 시도하고, 안되면 requests로 HTML 파싱해주세요."
```

**예상 시나리오 1 — 성공:**
- xlsx 파일 생성됨 → VSCode에서 파일 확인

**예상 시나리오 2 — 보안 팅김:**
```
에러: 403 Forbidden 또는 연결 거부
```
→ Claude Code에게: "보안 때문에 막혔대요. DOM 방식으로 바꿔주세요"

**예상 시나리오 3 — 계속 팅김:**
→ Claude Code에게: "자동화가 안되네요. 제가 수동으로 복사할게요. 복사한 텍스트를 엑셀로 정리하는 코드 짜주세요"
→ 웹사이트에서 원하는 텍스트 직접 복사
→ Claude Code에 붙여넣기 → 정리 요청

> **이 팅기는 경험 자체가 학습이다. 에러 메시지 읽는 법, 대안 찾는 법.**

### 3-3. 엑셀 파일 열어보기
1. 생성된 xlsx 파일 찾기 (VSCode 사이드바 또는 탐색기)
2. 더블클릭 → 엑셀로 열림
3. 데이터 들어있으면 성공

### 3-4. 실전 미션 (반복 3회)

**미션 1:** 원하는 쇼핑몰에서 상품명 + 가격 20개 → 엑셀
**미션 2:** 뉴스 제목 + 링크 30개 → 엑셀
**미션 3:** 원하는 사이트 직접 골라서 데이터 수집 → 엑셀

각 미션마다:
- API 먼저 시도
- 안되면 DOM
- 안되면 수동 복붙 + Claude 정리

## Phase 4 — 실전 B: OCR → 파일정리

### 4-0. OCR이란
사진/이미지 안의 텍스트를 읽어내는 것.
예: 명함 사진 → 이름/전화번호/이메일 추출 → 엑셀 정리

### 4-1. 실습 준비
사진 파일 하나 준비 (명함, 영수증, 문서 사진 등)
→ VSCode 프로젝트 폴더에 복사해 넣기

**파일 복사하는 법:**
1. 탐색기에서 사진 파일 찾기
2. 오른쪽 클릭 → 복사
3. `Documents/ai-실습` 폴더 열기
4. 붙여넣기

### 4-2. Claude Code에게 OCR 시키기
```
"image.jpg 파일에서 텍스트를 읽어주세요.
읽어낸 텍스트를 result.xlsx 파일에 정리해주세요."
```

### 4-3. 결과 확인 + 수정
1. result.xlsx 열기
2. 잘못 읽은 부분 확인
3. Claude Code에게: "3번째 행의 전화번호가 틀렸어요. 010-1234-5678로 수정해주세요"

### 4-4. 실전 미션

**미션 1:** 명함 사진 5장 → 이름/전화/이메일 엑셀 정리
**미션 2:** 영수증 사진 → 항목/금액 엑셀 정리
**미션 3:** 손글씨 메모 사진 → 텍스트 변환 → 문서 파일

## 자주 막히는 상황 대처법

| 상황 | 대처 |
|------|------|
| 터미널에 아무것도 안됨 | VSCode 껐다 켜기 |
| 명령어 쳤는데 에러 | 에러 메시지 전체 복사 → Claude Code에 붙여넣기 |
| 파일이 어디 있는지 모름 | VSCode 왼쪽 사이드바 확인, 또는 `ls` 입력 |
| 크롤링 계속 팅김 | 수동 복붙으로 전환 |
| Python 없다고 뜸 | Phase 0 - 0-4 다시 처음부터 |
