# teach-ai 조사내용

## 주제 정의

**"AI를 모르는 신입사원이 AI로 AI를 배우는 커리큘럼"**

터미널을 열고 하나하나 따라하면서 AI 기본 용어·기능을 익히는 방식.
즉, AI 도구 자체가 교사(tutor) 역할을 하면서 실습 중심으로 진행되는 형태.

---

## 1. 이미 존재하는 유사 커리큘럼/프로그램

### A. GitHub Copilot CLI for Beginners (가장 근접한 사례)

> "터미널을 열고 따라하면서 AI를 배운다"는 개념에 가장 가까운 공개 커리큘럼

- **형태**: 8챕터 무료 오픈소스 실습 코스 (GitHub 레포)
- **학습 방식**: 클론 → 터미널에서 Copilot CLI 실행 → 챕터별 미션 수행
- **구조**: 각 챕터마다 (1) 실생활 비유로 개념 설명 → (2) 기술 내용 → (3) 직접 실습
- **대상**: AI 경험 없는 완전 초보
- **특징**: GitHub Codespaces에서 바로 실행 가능 (로컬 환경 세팅 불필요)
- **비용**: 무료 (GitHub Copilot Free 계정으로 가능)
- **출처**: [GitHub Copilot CLI for Beginners](https://github.com/github/copilot-cli-for-beginners) / [Microsoft 공식 포스트](https://developer.microsoft.com/blog/get-started-with-github-copilot-cli-a-free-hands-on-course)

**챕터 구성**:
1. Setup & First Steps
2. Context and Conversations (@ 레퍼런스 활용)
3. Development Workflows (코드 리뷰, 리팩토링, 디버깅, 테스트)
4. Custom Agents 만들기
5. Skills (태스크별 AI 지시문)
6. MCP Servers 연결
7. 통합 워크플로우 구축

---

### B. Microsoft Generative AI for Beginners (GitHub)

- **형태**: 21레슨 무료 오픈소스 커리큘럼
- **대상**: 비개발자 포함 입문자
- **학습 방식**: 레슨별 개념 + 실습 노트북 (Jupyter)
- **언어**: 영어 (다국어 지원 일부)
- **출처**: [microsoft/generative-ai-for-beginners](https://github.com/microsoft/generative-ai-for-beginners)

---

### C. Microsoft AI For Beginners (GitHub)

- **형태**: 12주 24레슨 완전 무료 커리큘럼
- **포함 내용**: 퀴즈, 실습, 과제, 솔루션
- **출처**: [microsoft/AI-For-Beginners](https://github.com/microsoft/AI-For-Beginners)

---

### D. Learn Prompting (learnprompting.org)

- **형태**: 자기 주도형 웹 기반 학습
- **학습 방식**: 단계별 텍스트 레슨 + 퀴즈 + 챌린지 모드
- **특징**: Feynman 기법, 퀴즈 자동 생성, 코딩 챌린지 AI 생성
- **대상**: 완전 비기술자도 가능
- **출처**: [learnprompting.org](https://learnprompting.org/courses/introduction_to_prompt_engineering)

---

### E. Microsoft Credentials AI Challenge (2026)

- **형태**: 무료 시나리오 기반 마이크로 자격증
- **모듈**: AI 채팅 업무 자동화, AI 리서치 에이전트 보고서, Copilot Studio 에이전트 제작
- **특징**: 인터랙티브 랩 기반 평가 (실제 실행)
- **출처**: [Microsoft Learn AI Challenge](https://learn.microsoft.com/en-us/credentials/microsoft-credentials-ai-challenge)

## 2. 한국 기업 교육 현황

### 한국AI교육진흥원 (ai-campus.kr)

- 500여 기업·기관에 맞춤 기업 AI 교육 제공
- 삼성전자, 한화, GS칼텍스 등 대기업 레퍼런스
- ChatGPT/Gemini/Claude 실무 활용 실습 중심
- 자격증 과정 4종 (생성형AI실무전문가 등)
- **단점**: 터미널 기반 self-paced 방식은 아님. 강사 주도형
- **출처**: [ai-campus.kr](https://ai-campus.kr/)

- 목표: 전사 임직원 50%가 생성형AI 변화 이해 / 15%가 스스로 AI로 문제 해결
- 강의보다 실무 적용 중심으로 전환 중
- **출처**: [Comento HRD 포럼 리포트](https://blog.comento.kr/2026-generative-ai-education-milestone-execution-guide/)

## 3. 핵심 인사이트

### 현재 시장의 빈틈

| 구분 | 존재 여부 | 설명 |
|------|-----------|------|
| AI 개념 교육 (강의형) | ✅ 많음 | 유데미, 인프런, 한국AI교육진흥원 |
| AI 도구 사용법 (웹 기반) | ✅ 있음 | Learn Prompting, Microsoft Learn |
| 터미널 기반 AI 실습 (개발자용) | ✅ 있음 | GitHub Copilot CLI for Beginners |
| **비개발자 신입사원 대상 터미널 따라하기 AI 교육** | ❌ **없음** | **이 포지션이 비어있음** |

### 가장 근접한 롤모델: GitHub Copilot CLI for Beginners

- 터미널 열기 → 설치 → 실행 → 챕터별 미션의 구조가 정확히 원하는 방향
- 단, 대상이 개발자 (npm, Python 기본 지식 전제)
- **비개발자 신입사원용으로 한국화 + 난이도 조정하면 차별화 가능**

### AI가 교사 역할을 하는 방식의 실제 작동 원리

```
[신입사원]
   ↓ 터미널 열기
   ↓ Claude Code / Copilot CLI 설치
   ↓ AI에게 "나 AI 처음인데 뭐부터 해야 해?" 입력
   ↓ AI가 단계별 미션 제시
   ↓ 신입사원이 따라하며 용어/기능 체득
   ↓ 다음 챕터로 자동 이동
```

## 4. 커리큘럼 설계 시 참고할 구조 (Copilot CLI 방식 차용)

```
Day 1: 설치 & 첫 대화
  - 터미널이 뭔지 / CLI vs GUI 차이
  - Claude Code 또는 Copilot CLI 설치
  - 첫 프롬프트 입력 ("안녕, 나 AI 처음이야")

Day 2: 프롬프트 기초
  - 명령 vs 질문 vs 요청의 차이
  - 컨텍스트 주기 실습

Day 3: 업무 자동화 맛보기
  - 파일 정리, 이메일 초안, 보고서 요약

Day 4: AI 한계 이해
  - 환각(hallucination) 직접 경험
  - 검증 습관 만들기

Day 5: 나만의 워크플로우 설계
  - 내 업무에서 AI 쓸 수 있는 곳 3개 찾기
```
