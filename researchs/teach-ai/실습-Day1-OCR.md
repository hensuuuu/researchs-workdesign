# Day 1 실습 — 사진 → 엑셀 변환

> VSCode + Claude Extension 설치 완료 상태에서 시작

---

## Step 1. 작업 폴더 만들기

1. 바탕화면에서 빈 곳 **오른쪽 클릭**
2. "새로 만들기" → "폴더" 클릭
3. 폴더 이름: `ai-작업` 입력 → Enter

**확인:** 바탕화면에 `ai-작업` 폴더가 생겼으면 성공

---

## Step 2. 사진 파일 폴더에 넣기

1. 정리할 사진 파일 찾기 (명함, 영수증, 문서 등)
2. 사진 파일 **오른쪽 클릭** → "복사"
3. `ai-작업` 폴더 **더블클릭**으로 열기
4. 폴더 안 빈 곳 **오른쪽 클릭** → "붙여넣기"

**확인:** `ai-작업` 폴더 안에 사진 파일이 보이면 성공

---

## Step 3. VSCode에서 폴더 열기

1. VSCode 실행
2. 상단 메뉴 **File** 클릭
3. **Open Folder** 클릭
4. `ai-작업` 폴더 선택 → **폴더 선택** 버튼 클릭

**확인:** VSCode 왼쪽 사이드바에 사진 파일 이름이 보이면 성공

---

## Step 4. Claude에 사진 첨부하기

1. VSCode 왼쪽 사이드바에서 **Claude 아이콘** 클릭
2. Claude 채팅창이 열리면 성공
3. 채팅창 하단 **클립 아이콘(📎)** 클릭 → 사진 파일 선택

   > 클립 아이콘 없으면: 채팅창에 사진 파일을 **드래그해서 끌어다 놓기**

**확인:** 채팅창에 사진 미리보기가 뜨면 성공

---

## Step 5. Claude에게 요청하기

사진이 첨부된 상태에서 아래 문장을 **그대로 복사해서 붙여넣기**:

```
이 사진에서 텍스트를 전부 읽어줘.
읽은 내용을 표 형식으로 정리해서
result.xlsx 파일로 저장해줘.
파일은 지금 열려있는 폴더(ai-작업)에 저장해줘.
```

→ Enter 또는 전송 버튼 클릭

---

## Step 6. 결과 확인

1. VSCode 왼쪽 사이드바에 `result.xlsx` 파일이 생겼는지 확인
2. `result.xlsx` **더블클릭** → 엑셀로 열림
3. 내용이 들어있으면 성공

---

## OCR이 틀렸을 때 (백업 방법)

### 상황 A — 일부만 틀림
Claude 채팅창에:
```
3번째 행의 전화번호가 틀렸어. 010-1234-5678로 수정해줘.
```

### 상황 B — 사진이 너무 흐려서 못 읽음
1. 사진을 핸드폰으로 다시 찍기 (더 밝은 곳에서, 흔들리지 않게)
2. Step 4부터 다시

### 상황 C — xlsx 저장이 안됨
Claude 채팅창에:
```
xlsx 저장이 안되면 csv 파일로 저장해줘.
파일 이름은 result.csv로.
```
→ csv 파일 열기: 더블클릭 → 엑셀로 열림
