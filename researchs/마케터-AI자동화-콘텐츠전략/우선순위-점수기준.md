## 기준 2. 바이럴 포맷 적합도 — 20점

각 항목 충족 시 4점, 5개 항목 합산:

| 체크 항목 | 판단 기준 |
|---------|---------|
| (a) 구체적 숫자 포함 | 제목/썸네일에 숫자 1개 이상 |
| (b) Before/After 또는 비교 구조 | 전후 비교 또는 A vs B 구조 |
| (c) 따라할 수 있는 워크플로우 | 단계별 실행 가이드 포함 |
| (d) 실제 결과/성과 명시 | 검증된 수치 또는 직접 경험 |
| (e) 시간/비용 절약 명시 | 절약량 또는 효율 수치 포함 |

**근거:** 숫자+결과형 제목 유튜브 조회수 x4.8배, LinkedIn 캐러셀 +596%, 인스타 완주율 90%+ 릴스 알고리즘 부스트

## 기준 6. 제작 난이도 (더미 대체 가능성) — 15점

실제 데이터(광고 계정, GA4, CRM 등) 없이 더미 데이터로 콘텐츠 시연이 가능한 정도.

| 조건 | 점수 |
|------|------|
| 더미 데이터를 AI가 즉시 생성 가능 (프롬프트만으로 시연 완결) | 15점 |
| 더미 데이터 필요하지만 직접 만들기 쉬움 (간단한 CSV, 텍스트) | 12점 |
| 더미 가능하지만 구조가 복잡해 준비 시간 필요 | 8점 |
| 실계정 필요하지만 무료 테스트 계정으로 대체 가능 | 5점 |
| 실계정·실데이터 없으면 시연 불가 (GA4 연동, 광고 계정 등) | 2점 |
