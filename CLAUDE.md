# 리서치 & 기획 규칙

## 근거 수집 규칙

1. **내부 데이터만으로 판단 금지** — 반드시 외부 출처를 기반으로 근거를 가져올 것
2. **최소 근거 5개 이상** — 단일 출처 의존 금지, 다양한 출처에서 수집
3. **출처 링크 포함 필수** — 모든 근거에 URL을 명시할 것
4. **최신 정보 우선** — 근 3개월 이내 발행된 정보를 우선 사용, 오래된 정보는 날짜 명시

## 검색 강제 규칙 (위반 금지)

5. **수치/순위/경쟁도 판단 시 반드시 직접 검색** — 추론·추정·내부 지식으로 수치를 만들어내는 것은 엄격히 금지. 검색량, 콘텐츠 수, 경쟁도, 시장 규모 등 모든 수치는 WebSearch로 직접 확인 후 기재할 것
6. **"간접 추론" 표현 사용 금지** — "추정", "~로 보임", "~일 것으로 예상" 등의 표현으로 수치를 제시하는 것 금지. 검색하지 않았으면 수치를 쓰지 말 것
7. **검색 없이 점수 매기기 금지** — 점수표, 우선순위, 순위 작성 시 각 항목의 근거를 검색으로 확인한 후에만 점수 부여
